# Điện Lạnh Hậu Phương

Website giới thiệu dịch vụ điện lạnh Hậu Phương, cung cấp các giải pháp sửa chữa, bảo trì và lắp đặt hệ thống điều hòa không khí.

---

## Giới thiệu

Dự án được xây dựng nhằm:

- Giới thiệu dịch vụ một cách trực quan
- Cung cấp thông tin rõ ràng cho khách hàng
- Đảm bảo trải nghiệm tốt trên nhiều thiết bị

---

## Cấu trúc thư mục

```
├── index.html        # Trang chủ
├── css/
│   ├── style.css     # CSS chính
│   └── data.css      # CSS bổ sung
├── img/              # Thư mục hình ảnh
```

---

## Công nghệ sử dụng

- HTML5
- CSS3
- Bootstrap 5.3.2
- Font Awesome 6.6.0
- JavaScript

---

## Tính năng

- Giao diện responsive trên nhiều thiết bị
- Thanh điều hướng thân thiện với mobile
- Thiết kế đơn giản, dễ sử dụng
- Tối ưu hiển thị và trải nghiệm người dùng

---

## Cách sử dụng

1. Clone repository:

```bash
git clone <your-repo-link>
```

2. Mở file `index.html` bằng trình duyệt
