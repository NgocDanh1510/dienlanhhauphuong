# Điện Lạnh Hậu Phương

Trang web dịch vụ điện lạnh Hậu Phương - cung cấp các dịch vụ sửa chữa, bảo trì và lắp đặt hệ thống điều hòa không khí.

## Cấu trúc dự án

- index.html - Trang chủ
- css/style.css - CSS chính
- css/data.css - CSS dữ liệu
- img/ - Thư mục hình ảnh

## Công nghệ sử dụng

- HTML5
- CSS3
- Bootstrap 5.3.2
- Font Awesome 6.6.0
- JavaScript

## Tính năng

- Giao diện responsive
- Thanh điều hướng di động
- Bố cục hiện đại
- Tối ưu hóa cho các thiết bị khác nhau

## Cách sử dụng

1. Mở tệp index.html trong trình duyệt web
2. Điều hướng qua các trang bằng menu navigation
